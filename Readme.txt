----------------------------------------------------
AGE OF EMPIRES I

This game has been tested on Windows 7 and 10 and it's fully working.

Instructions:
* double click AOESETUP.EXE and launch the Install program
* select a folder and choose Typical Installation (not FULL)
* select Play Age of Empires
* if you some Drivers errors just ignore them and click ok, after some
  warnings, the game will start
